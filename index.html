🚀 BySamIO Helm Charts

Official Helm charts repository for CasePack and BySamIO infrastructure

⚡ Quick Start

Traditional Helm repository:

helm repo add bysamio https://bysamio.github.io/charts/
helm repo update
helm search repo bysamio

OCI registry (Helm 3.8+):

helm install my-release oci://ghcr.io/bysamio/charts/<chart> --version <version>
